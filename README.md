Expense Tracker — Трекер расходов

Автор: Орлова Ксения 


Описание программы

Expense Tracker — это десктопное приложение на Python для учёта личных расходов.  
Позволяет добавлять траты по категориям, фильтровать по дате и категории, считать сумму за выбранный период, а также сохранять и загружать данные в JSON.

Как использовать

1. Установите Python 3.x.
2. Запустите файл `expense_tracker.py`.
3. В верхней панели введите:
   - Сумму (положительное число)
   - Категорию (выберите из списка)
   - Дату (в формате ГГГГ-ММ-ДД)
4. Нажмите «Добавить расход»— запись появится в таблице.
5. Для фильтрации:
   - Выберите категорию и/или введите дату в блоке **Фильтры**
   - Нажмите "Применить фильтр»
   - Нажмите «Сбросить», чтобы увидеть все записи
6. Для подсчёта суммы за период:
   - Введите начальную и конечную дату
   - Нажмите «Подсчитать за период"
7. Все данные автоматически сохраняются в `expenses.json`.
